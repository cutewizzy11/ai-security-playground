# Code of Conduct

This project follows a simple code of conduct inspired by the Contributor Covenant.

- Be respectful and inclusive.
- Assume good faith.
- No harassment, discrimination, or personal attacks.
- Use welcoming and inclusive language.
- Focus on constructive feedback.

Maintainers may take appropriate action, including warning or blocking users, in response to unacceptable behavior.

If you experience or witness unacceptable behavior, please open an issue (if comfortable) or contact a maintainer privately if contact information is provided in the repository description.
