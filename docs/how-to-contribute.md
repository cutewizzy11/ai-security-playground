# How to Contribute

This document gives more detail on how to contribute to AI Security Playground.

- Start by reading `README.md` and `CONTRIBUTING.md`.
- Look for issues labeled `good first issue` if you are new.
- When adding new material, prefer small, focused PRs over huge ones.
- For new tracks or projects, briefly explain your idea in an issue before you start coding.

Thank you for helping build this learning and experimentation space!
