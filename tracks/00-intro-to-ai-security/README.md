# Track: Intro to AI Security

This track introduces core ideas in AI security for beginners.

## Goals

- Understand what AI security is and why it matters.
- See concrete examples: adversarial examples, phishing detection, basic anomalies.
- Prepare contributors to explore more advanced tracks.

## Contents

- Notebooks explaining key concepts in an accessible way.
- Simple, well-commented Python code using common libraries.

- Improve explanations and visualizations.
